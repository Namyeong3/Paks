App class list:

 Main:
 Where the main code in the app executes, classes are called and key bindings are.

 Textbox:
 A GUI element for text.

 Button:
 A clickable GUI element containing a Textbox.

 Menu:
 A GUI element containing a list of buttons.

 Tab:
 A GUI element that occupies the whole screen containing multiple UI element.

 LoadingElement:
 A GUI element displaying a loading animation.

 StoryData:
 A data file with the tree data of the story.

 StoryLoader:
 A class that deals with loading the story data from the file to the memory.
